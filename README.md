# booker

personal book‑tracking / reading‑list app — **(work in progress)** . don’t expect much yet.

## what it is

a backend (python / fastapi) + React frontend web app to track books: what you read, plan to read, maybe notes/metadata.  

## status

- no stable release — repo mostly a skeleton / early draft  

## planned features

- add book entries (title, author, status, maybe more metadata)
  
- list / filter books (read, to‑read etc.)
    
- user accounts, book metadata
